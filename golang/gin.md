# 介绍gin
```
gin 是一个高性能，轻量级的web框架，特别是它的支持的中间，使用起来特别灵活，它还支持数据校验等
```
# 中间使用场景
```
日志
认证(身份校验等)
跨域检查
```
一些知识
```
gin 为每一个请求，都会分配一个协程处理
gin 内置了多种模板引擎，c.Json,c.Html等，
    还可以自定义模板引擎gin.SetHTMLTemplate()
gin 如何处理err
    一般在中间件里处理
        解析err对象，获取对应的状态码，错误code，错误信息等
    可以调用c.Error(err)将err存放到当前context中
```
